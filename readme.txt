This plugin provides a quick and easy way to add image with lightbox block using Gutenberg visual editor.

== Description ==

Image with Lightbox Block for Gutenberg provides a block for the new Gutenberg editor in WordPress which allows you to add images with lightbox functionality and basic styling options.

This plugin allows you to add images that open in a lightbox when clicked.

You can easily customize image settings like size, alignment and lightbox behavior.

== Installation ==

From the admin panel, Go to your WordPress Admin -> Plugins -> Add New. Search for Image with Lightbox Block For Gutenberg. Install and Activate.

From directories, Upload `Image with Lightbox Block For Gutenberg` to the `/wp-content/plugins/` directory and activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How to add Image with lightbox?  =

After installing Plugin, you need to go to page or post section where you can see the Gutenberg editor and click on add block. Select Image with LightBox Block and add add image and then enable option from side panel.
